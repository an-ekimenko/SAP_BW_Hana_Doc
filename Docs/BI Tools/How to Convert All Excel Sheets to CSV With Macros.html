How to Convert All Excel Sheets to CSV With Macros

  • Small Business
    • |
  • Accounting & Bookkeeping
    • |
  • Excel
By Darrin Koltow

Related

  • Recovering Deletions in Excel

Excel's Save As command lets you save a single worksheet in the CSV (comma seperated value) format, but not an entire workbook. Saving individual sheets by hand is a tedious project for large workbooks, but a cinch for a Visual Basic macro. The specific function for saving a particular worksheet in CSV, or several other formats, is the Worksheet object's SaveAs function. This function takes several arguments, but you need only two to save a worksheet as CSV. The first is Filename, and the second, FileFormat, enables you to specify CSV as the format to save in.

1

Create a new workbook in Excel, which will contain your macro, then press "Alt-F11" to open the Visual Basic integrated development environment. Double-click the "This Workbook" item in the navigation pane at left to open a code window in which to enter macro code.

2

Paste in the following macro, which saves all sheets of the open workbook as CSV files. The Dim statements create variables that the macro needs to run. The DisplayAlerts statement prevents Excel from asking if you want to overwrite an existing CSV file, when you run the macro. The remaining statements in the macro iteratively save each worksheet to the "C:\Tmp" folder.

Sub saveSheetsAsCSV() Dim i As Integer Dim fName As String Application.DisplayAlerts = False For i = 1 To Worksheets.Count fName = "C:\Tmp\MySheet " & i ActiveWorkbook.Worksheets(i).SaveAs Filename:=fName, FileFormat:=xlCSV Next i End Sub

3

Return to Excel and click the View menu's "Hide" command, which is in the Window group, to hide the workbook. Hiding the workbook will prevent you from accidentally entering data in it. The workbook's sole purpose is to hold your macro, not data.

4

Close Excel, which will ask if you want to save the workbook in which you just entered the macro. Click "Yes," then save the workbook with the following filename, replacing "" with the name of your Windows user account. Excel will load this file automatically the next time it starts.

C:\Users\\AppData\Roaming\Microsoft\Excel\XLSTART\personal.xlsb

5

Restart Excel, which will load the hidden workbook containing the CSV macro you just wrote. Open any existing workbook with data on more than one worksheet, then press "Alt-F8" to display a list of macros. Double-click the "saveSheetsAsCSV" item to run your macro.

6

Open the files in the "C:\Tmp" folder whose names start with "MySheet" and end with ".CSV". This step enables you to verify that your macro successfully converted each worksheet into a CSV file.

How to Read the Data From an Excel Sheet Using VB Macros Coding

By Ryan Menezes

Related

  • How to Output CSV Files in VB6
  • How to Start a Macro When Opening an Excel Spreadsheet
  • How to Read an Element or Attribute in Java XML JDOM
  • How to Copy XLS Files to SPSS

Excel sheets read data in cells using cell references, which are the addresses that identify each cell. Visual Basic macros also read data using cell references, extracting data by reading the cell's Value property. This lets macros run scripts using the cells' contents. For example, a macro might validate the data in an employee timesheet. The macro can read a cell, check whether it exceeds a maximum amount and open a dialog box with an error message if the figure is too large.

1

Click "Developer" in Excel's menu bar.

2

Click the Visual Basic icon in the ribbon's Code tab to launch the Visual Basic for Applications window.

3

Select the macro in the window's Project box that you want to read Excel's data.

4

Type the following line of code:

Dim contents As Integer

This declares a variable to store the cell's data.

5

Replace "Integer" with "String" if the cell contains a string. Replace "Integer" with "Date" if the cell contains a date.

6

Add the following line, which extracts a cell's data:

Contents = Range("A1").value

7

Replace "A1" with the address of the cell whose data the macro must read. Visual Basic will store the cell's contents in the variable.

How to Save a File in CSV Format in Microsoft Office 2007

By Lori Kaufman

Related

  • How to Open a Zip File in Excel 2007
  • How to Vertically Split Cells in PowerPoint 2003
  • How to Open an XLR Document
  • How to Read an XLS File Without Excel
  • How to Expand and Collapse in Excel 2007

You can insert or generate the data for the CSV file in Microsoft Office Excel 2007 and then save the file in CSV format. CSV files outputted by Excel 2007 can be used in any program that can read this format. If the external program does not support the default Excel format, XLSX, you can generate the data in Excel faster than you would generate it manually. As long as your other program can read CSV files, you can import and use the data.

1

Launch Microsoft Office Excel 2007 and open the spreadsheet that you wish to save in CSV format. Use the "Ctrl-O" keyboard shortcut to open the file faster.

2

Click the Office button in the top left corner of the Excel window and click the "Save As" button.

3

Select the folder where you want to store the CSV file and enter a name for the file in the File Name field.

4

Select "CSV (Comma delimited) (*.csv)" in the Save As Type drop-down box and then click "Save" to save the file as a CSV file.

5

Click "OK" when Excel warns you that only the selected sheet will be saved.

6

Click "Yes" when Excel warns you that certain features in your workbook may not be compatible with CSV format. The CSV file is created in the folder you selected earlier.

References

  • Microsoft Office: Import or Export Text (.txt or .csv) Files

Resources

  • Massey University, New Zealand: Microsoft Office 2007 Training Manual [PDF]

Tips

  • Only the selected sheet is saved as a CSV file; repeat the process for each sheet if your workbook contains multiple sheets and you want to export all of them.

How to Import Data From MS Word to MS Excel

By Filonia LeChat

Related

  • How to Export XLS Files to Word
  • How to Split a Table in Word
  • How to Save a File in CSV Format in Microsoft Office 2007
  • How to Enable Scrolling in Excel 2007
  • How to Import QuickBooks Data to Microsoft Access

Microsoft Office is designed so you can import data from one Office program to another seamlessly. For example, you can import Word document data and even whole Word documents in their entirety into your Excel spreadsheets. With a few clicks, you can bring text from a catalog of product descriptions in Word into your Excel spreadsheets on pricing and sales or lift tables you produced in annual reports or employee salary surveys right off the page, saving you from starting from scratch.. You can even make changes to the Word data after it is imported into Excel.

Single Cell Import

1

Launch Microsoft Excel and open the Excel spreadsheet you want to import Word document data into.

2

Click the Insert tab, then click the Object button on the Ribbon.

3

Click the Create from File tab on the Object window and browse to the Microsoft Word document to import.

4

Double-click the name of the file and click OK to close the Object window. You are brought back to the Excel spreadsheet with the Word document merged into it. Note the cell text box below the ribbon shows =EMBED(Document,).

5

Double-click anywhere within the Word document to open it, making it editable. You can perform all Word features here, even though you are still in Excel, such as changing or reformatting text.

6

Click the File tab and select Save As. Rename the current Excel spreadsheet instead of saving over it so your nonmerged version remains accessible. Click the Save button.

Cell by Cell

1

Start Microsoft Word and open the document with the table of data to import into Excel.

2

Click the small hatch mark in the table's top-left corner, which highlights the table in default Word blue, then press "Ctrl-C" to copy the table. Close Word if desired.

3

Open Excel. Click the cursor into the first cell in the grid and press "Ctrl-V" keys to paste in the Word table's contents. The data exports from Word into Excel, with the same cell layout. For example, if your Word data was a table of two rows with three columns, the Excel spreadsheet now shows two rows with three columns.

References

  • Microsoft Office: Copy a Word Table into Excel

Tips

  • In the single-cell import process, your Microsoft Word document is pasted into a single Excel cell. Excel resizes the grid to fit the cell, which will be larger and show your document within the grid. This process is ideal for importing whole Word documents, such as proposals, and you do not have to open Microsoft Word to perform this task. The cell-by-cell import process works best for tables within Word. You can edit the table's contents, if desired, after pasting into Excel.

How to Make a List of Excel Data Into Data Separated by Comma

By Shawn McClain

Related

  • How to Recover Lost Excel 2007 Data From an Unsaved Worksheet
  • How to Reconfigure a Horizontal Row to a Vertical Column in Excel
  • How to Hide Empty Data in Excel Graphs
  • How to Fill Colors in a List in Excel
  • How to Make Sortable Headings in Excel

Microsoft Excel 2010 stores data within cells, which are simply segregated pieces of the spreadsheet that Excel can then reference to work with the value in the cells. Since not every data-manipulation program can understand an Excel file structure, the program allows you to save your data in a number of different formats, including the comma-separated values format (CSV). Saving a set of data in this format will removing any special Excel formatting from the data and simply place it in a list, with a comma separating each piece of data. CSV files are popular because many types of programs, including most spreadsheet and word processing programs, can open them.

1

Double-click the icon for the Excel 2010 workbook that you want to work with to open.

2

Click on the top cell in your list of Excel data and hold down the mouse button. Drag the mouse button down until you reach the last cell in the list. Press "Ctrl" and "C" to copy this information to your clipboard.

3

Click on "Sheet2," located at the very bottom of your spreadsheet, to bring up a blank worksheet.

4

Right-click on cell "A1" and move your mouse over "Paste Special." Choose "Transpose" from the paste options that appear. The "Transpose" icon will look like a set of vertical and horizontal cells with an arrow running between them. Once you click this option, your data will be pasted to the new worksheet, but it will be orientated horizontally instead of vertically. This will allow the CSV file to separate the data by commas, instead of by lines.

5

Click on the "File" tab at the top of the screen and choose "Save As" from the list of options that appear. Type in a name for the file in the "File name" field, then click the drop-down arrow next to "Save as type." Choose "CSV (Comma delimited)" from the list of options. Be careful that you don't choose "CSV MS-DOS" or "CSV Macintosh."

6

Click "Save" to close the window. Click "OK" when the pop-up window appears, which explains that only the current worksheet will be saved. Then click "Yes" on the next pop-up, which explains that some formatting will not be able to be saved to the CSV file type.
